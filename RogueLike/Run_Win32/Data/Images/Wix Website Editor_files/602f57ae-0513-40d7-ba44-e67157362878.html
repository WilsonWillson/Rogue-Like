      Game Overview

      Utility Based: Behavior System

      The game AI is governed by a simple utility system, which determines which of the Behaviors each monster will attempt. These behaviors are chosen from the given XML (more on that in the XML Data Driven Page). Going through each behavior, we calculate which Behavior fits the situation best, based on conditions such as, Player's Range to the enemies home, Enemies health, Enemies range away from the player, and wether or not the enemy has "minions" or similar monsters around him.

      The different behaviors currently implemented for the A.I. our:

      • Attack

      • Leader

      • FollowLeader-

      • Flee

      • Pursue (The Player)

      • Spawner- Spawns enemies, enemy type is chosen by data from the XML.

      • Teleport

      • Wander- Wanders the map around his home location.
